# **App Name**: ShangLudo Arena

## Core Features:

- Game Lobby: Allow players to view available game rooms, filter by play mode and join existing games. Player will use the email provided in prompt to allow friends to find games
- Multiple Play Modes: Implement classic Ludo and quick Ludo modes, with custom rulesets available to spice up the gameplay.
- Dice Roll Animation: Animate the dice roll to give a realistic feel with clear visuals and sound effects when a dice is rolled and plays the numbers.
- In-Game Chat: Enable real-time chat functionality during the game, allowing players to communicate with each other using text or emojis.
- Game Logic Enforcement: Implement core game logic and enforce fair play.
- AI Opponent (Optional): Add an AI opponent that a player can play to when they do not have an opponent in online play. Note: This feature will not persist game status in any database.

## Style Guidelines:

- Primary color: Deep, saturated blue (#3F51B5) to evoke a sense of strategy, intellect, and board game tradition.
- Background color: Very light, desaturated blue (#F0F2FA) to provide a clean, non-distracting backdrop that allows the colorful game pieces and interface elements to pop.
- Accent color: A vibrant orange (#FF9800) to highlight interactive elements, indicate player turns, and draw attention to key actions.
- Body and headline font: 'Inter' (sans-serif) for a modern, neutral, and highly readable appearance. Well-suited for on-screen text in both headlines and body.
- Use flat design icons that are simple and recognizable. Consider icons related to dice, pawns, chat bubbles, and game settings.
- Ensure the Ludo board is the centerpiece of the UI, with all interactive elements readily accessible without cluttering the screen. Chat window will docked to the side of the game.
- Use subtle animations for dice rolls and piece movements to enhance the user experience without being distracting. Special effects could highlight winning moments.
